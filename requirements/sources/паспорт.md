Паспорт проекта
Название игры: «Русский язык: тренировочный полигон»
Цель: Систематическая отработка и закрепление знаний по грамматике, орфографии и пунктуации русского языка у школьников через интерактивные игровые задания.
Задача
•	обеспечить поэтапное освоение тем по русскому языку (от простого к сложному);
•	сформировать устойчивые навыки применения правил на практике;
•	мотивировать учащихся к повторению и углублению знаний через систему баллов, достижений и рейтингов;
•	выявить и проработать индивидуальные пробелы в знаниях.
Правила игры
1.	Игрок выбирает тему или проходит последовательный курс.
2.	ИИ генерирует от 5 до 20 заданий по выбранной теме.
3.	На каждое задание даётся время в зависимости от типа (выбор: 45 сек, ввод: 90 сек, анализ: 180 сек). При превышении ответ не засчитывается (не нарушает серию, подсказка не расходуется).
4.	За правильный ответ — +10 баллов; за неправильный — 0 баллов + пояснение правила.
5.	За серию из 3 и более верных ответов подряд — каждый ответ начиная с 4-го приносит +5 бонусных баллов. Итого: с 4-го верного подряд каждый даёт +15 баллов (базовые 10 + бонус 5). При неверном ответе серия обнуляется.
6.	Для перехода на следующий уровень нужно набрать ≥80 % правильных ответов.
7.	Доступно 2 подсказки на уровень (стоимость: −5 баллов ДО получения ответа). Если ответ верный — итого +5 баллов. Если неверный — 0 баллов, подсказка потрачена.
8.	Можно перепройти уровень для улучшения результата (баллы не суммируются, статистика обновляется, серия сбрасывается).
9.	Запрещён автоматический ввод ответов (работает антибот-система: детект быстрых ответов, повторяющихся паттернов, требование CAPTCHA при подозрении, временная блокировка при нарушениях).
Целевая аудитория
Ученики 8–11 х классов общеобразовательных школ РФ; взрослые, желающие повторить правила русского языка.
Реализация (на стадии обсуждения)
•	Веб приложение (основная версия): HTML5, CSS3, JavaScript, бэкенд на Node.js/Python.
•	PWA (мобильная версия): прогрессивное веб приложение с офлайн доступом к базовым заданиям.
•	Терминал ОС: консольная версия на Python/Go для Linux/macOS/Windows.
•	База данных: PostgreSQL/MongoDB (хранение статистики, заданий, профилей).
•	ИИ модуль: генерация заданий на основе правил и языковых корпусов.
•	Интеграция: API YandexGPT для формирования заданий и пояснений.
Возникшие проблемы при реализации
1.	Генерация разнообразных заданий — необходимость избежать повторов и обеспечить покрытие всех подтем.
2.	Адаптация сложности — подбор уровня заданий под индивидуальные знания игрока.
3.	Проверка открытых ответов (например, расстановка знаков препинания) — требуется точный морфологический и синтаксический анализ.
4.	Кросс платформенность — синхронизация прогресса между веб, PWA и терминалом.
5.	Античит — предотвращение автоматического ввода и накрутки баллов.
6.	Локализация интерфейса — единообразие терминов и формулировок в заданиях.
7.	Производительность ИИ — задержка при генерации заданий через API YandexGPT.
Краткое описание
Образовательный тренажёр по русскому языку в игровой форме. Игрок проходит уровни по темам (словосочетания, предложения, орфография и др.), выполняет задания, набирает баллы и открывает новые уровни. Доступны три платформы: веб, мобильное PWA и терминал. ИИ генерирует задания, проверяет ответы и даёт пояснения.
Механика игры
1.	Выбор темы/курса — меню с перечнем тем или последовательный трек.
2.	Генерация заданий — ИИ создаёт 5–20 заданий разного типа:
    o	выбор варианта из нескольких;
    o	ввод ответа с клавиатуры;
    o	сопоставление элементов;
    o	анализ текста (выделение основ, типов связей и т. п.).
3.	Выполнение и проверка — игрок вводит ответ, система мгновенно оценивает, начисляет баллы и показывает пояснение при ошибке.
4.	Прогресс и награды — накопление баллов, открытие уровней, достижения, ранги, сертификаты.
5.	Статистика — отчёт по темам, процент верных ответов, время выполнения.
6.	Повторение — возможность перепройти уровень для улучшения результата.
Язык разработки
•	Фронтенд (веб/PWA): TypeScript + Web Components (без тяжелых фреймворков), сборка esbuild/Vite.
•	Бэкенд API: Rust (axum, tokio) для высокой производительности и надежности.
•	ИИ сервис: Python 3.12 (asyncio, aiohttp) для интеграции с YandexGPT и NLP.
•	База данных: MongoDB (основная) для документов + Redis (кеш, таймеры) + Qdrant (векторные эмбеддинги).
•	Инфраструктура: Docker-compose (локально), Kubernetes/YaCloud (production).
•	Терминал: опционально для v2.0 (не входит в MVP).
Использование ИИ (YandexGPT через API)
•	Генерация заданий — создание уникальных примеров по заданной теме и уровню сложности.
•	Проверка открытых ответов — анализ синтаксиса, орфографии, пунктуации.
•	Пояснения к ошибкам — формулировка правил и примеров на естественном языке.
•	Адаптация сложности — подбор заданий на основе статистики игрока.
•	Автоматическое составление текстов для анализа (например, предложений с пропущенными знаками препинания).
